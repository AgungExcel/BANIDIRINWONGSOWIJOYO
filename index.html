Kode Google Apps Script (Backend)

PENTING: Karena sistem sebelumnya tertahan kebijakan peramban, kode di bawah telah diperbarui. Silakan ulangi langkah ini.
  1. Buka Google Sheet database Anda, lalu klik menu Ekstensi > Apps Script.
  2. Hapus seluruh kode lama yang ada di dalam editor tersebut.
  3. Klik tombol Copy Kode di bawah ini, lalu Paste ke dalam editor Apps Script Anda.
  4. Langkah Wajib: Di menu atas Editor Apps Script, pastikan nama fungsinya adalah setup, kemudian tekan tombol ▶ Jalankan (Run).
  5. Jika muncul, klik Tinjau Izin (Review Permissions) > Pilih Akun Anda > Lanjutan (Advanced) > Klik tulisan Buka Project (Tidak Aman) > Klik Izinkan (Allow).
  6. Klik Terapkan (Deploy) > Deployment Baru > Pilih jenis Aplikasi Web (Web App) > Akses: Siapa saja (Anyone) > Klik Terapkan.
  7. Copy URL yang dihasilkan, lalu masukkan ke dalam kolom Link Tulis Data (URL Google Apps Script) di menu Pengaturan aplikasi ini.
